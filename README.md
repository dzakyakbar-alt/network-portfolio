## 🧪 Experience

### IT Support → Network Engineer (Transition Role)

- Menangani troubleshooting harian terkait perangkat IT seperti laptop, aplikasi, printer, dan sistem operasi Windows.
- Memberikan support langsung kepada user untuk memastikan operasional kerja berjalan lancar.

### Network Troubleshooting
- Melakukan analisis gangguan jaringan seperti koneksi tidak stabil, RTO, packet rejected dan packet loss.
- Menggunakan tools seperti ping dan traceroute untuk mengidentifikasi jalur jaringan dan sumber masalah.
- Menangani permasalahan koneksi antar cabang hingga jaringan kembali normal.
- Melakukan perbaikan dan peningkatan infrastruktur jaringan, termasuk penggantian perangkat dan perbaikan kabel LAN.
- Melakukan troubleshooting end-to-end dari perangkat hingga konektivitas jaringan

### MikroTik Configuration
- Melakukan konfigurasi dasar hingga menengah pada router MikroTik melalui Winbox.
- Mengelola DHCP Server, Firewall, dan routing.
- Melakukan pengecekan interface, routes, dan traffic untuk troubleshooting.

### VPN & Connectivity
- Memahami dan menganalisis koneksi VPN menggunakan PPTP, L2TP, SSTP Tunnel.
- Troubleshooting koneksi antar site melalui tunnel.

### Load Balancing & Failover
- Memahami dan mengimplementasikan konsep load balancing menggunakan metode PCC pada MikroTik.
- Mengelola lebih dari satu ISP untuk membagi beban trafik jaringan.
- Mengkonfigurasi failover ISP untuk memastikan koneksi tetap berjalan saat salah satu jalur internet mengalami gangguan.
- Menggunakan metode pengecekan gateway (check-gateway, ping) untuk memastikan perpindahan jalur berjalan otomatis.
- Melakukan analisis jalur routing menggunakan traceroute untuk memastikan trafik berjalan sesuai rule yang dibuat.

### Monitoring & Network Visibility
- Menggunakan Cacti untuk monitoring jaringan berbasis SNMP.
- Menganalisis kondisi device berdasarkan status monitoring (up/down, traffic).

### Routing & Network Design
- Memahami konsep routing dinamis menggunakan OSPF untuk pertukaran routing antar network.
- Memahami penggunaan RIP sebagai routing protocol sederhana dalam jaringan kecil.
- Mempelajari perbedaan penggunaan routing statis dan dinamis dalam berbagai skenario jaringan.
- Menggunakan routing untuk mendukung konektivitas antar jaringan dan antar cabang.

### Initiative & Problem Solving
- Secara mandiri menangani permasalahan jaringan tanpa selalu bergantung pada Network Engineer.
- Berinisiatif mempelajari konfigurasi dan konsep jaringan secara langsung dari kasus nyata.
- Berhasil menyelesaikan berbagai kendala jaringan hingga kembali normal.
